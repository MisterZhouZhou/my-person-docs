# 开发实用技巧
## 1、label展示两行超出...
```shell
UILabel *titleLab = [[UILabel alloc] initWithFrame:CGRectMake(0, 100, 200, 40)];
titleLab.numberOfLines = 2;
titleLab.lineBreakMode = NSLineBreakByTruncatingTail;
```
## 2、tableView滚动到顶部生效
```shell
  [self.tableView reloadData];

```
