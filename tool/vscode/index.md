## VSCode简介
`VSCode`全称`Visual Studio Code`是Microsoft在2015年4月30日开发者大会上正式宣布的项目。VSCode是一个运行于 **Mac OS X**、**Windows** 和 **Linux** 之上，针对于编写现代 Web 和云应用的跨平台源代码编辑器, 拥有强大的扩展功能和丰富的插件。
