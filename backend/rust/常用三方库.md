# 常用三方库
## 1、cargo-watch
### 1.1、安装
地址： https://crates.io/crates/cargo-watch

```shell
cargo install cargo-watch
```

### 1.2、使用
```shell
cargo watch -q -c -w src -x run
```

### 1.3、查看帮助
```shell
cargo watch --help
```
